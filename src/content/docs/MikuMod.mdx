[MikuMod](https://vocadb.net/Profile/MikuMod) is an automod for VocaDB with various features such as:

1. Sending welcome messages
2. Checking new edits
3. Fixing or reporting rule violations
4. Re-checking [entry reports](/docs/entry-reports) (based on wiki links)

Mirrored source code is available at https://github.com/Shiroizu/MikuMod-mirror

The current rule support status is tracked on the [rule table](/rules/table).

Direct messages to MikuMod are automatically forwarded to a private mod Discord channel.
