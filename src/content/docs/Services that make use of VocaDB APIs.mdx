## Album/song taggers

- [mp3tag script](https://community.mp3tag.de/t/ws-vocadb-net/17192): A script designed for tagging entire albums using the mp3tag application.
- [Vocatag](https://github.com/szc126/vocatag): A tool for tagging your music collection efficiently.
- [beets-vocadb](https://github.com/prTopi/beets-vocadb): Plugins for beets that use VocaDB, UtaiteDB, and TouhouDB for automatic tagging
