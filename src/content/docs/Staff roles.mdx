### Moderators

List of moderators: https://vocadb.net/User?groupId=Moderator

Responsibilities:

- Enforcing [rules](/rules)
- Enforcing [discussion rules](/docs/discussion-rules)
- Promoting users to [trusted users](/docs/user-groups#trusted-user)
- Promoting [trusted users](/docs/user-groups#trusted-user) to [moderators](/docs/user-groups#moderator)
- Simple wiki content updates/fixes
- Handling [entry reports](/docs/entry-reports)
- Handling [artist verification requests](/docs/artist-verification)
- Handling simple [content removal requests](/docs/content-removal-guidelines)
- Handling user account deletion requests
- Maintaining tag mappings (Niconico tag -> VocaDB tag)
- Handling spammers (with [StopForumSpam](https://www.stopforumspam.com/))

### Trusted users

List of trusted users: https://vocadb.net/User?groupId=Trusted

Responsibilities (optional):

- Handling regular [entry reports](/docs/entry-reports)
- Participating in rule discussions
