This page lists all relevant website updates starting from **May 2023**.
It can also be viewed on the [VocaDB Discord server](https://discord.com/invite/3bwXQNXKCz).

For information on the feature requests and bug reports, see the [VocaDB feature requests](https://wiki.vocadb.net/wiki/51/vocadb-feature-requests).

## 15. May 2023

**Improvements:**
* It is now possible to search for albums using their crossfade URLs ([#1321](https://github.com/VocaDB/vocadb/issues/1321))
* Translations are now cached, eliminating text "flickering" on most pages ([#1022](https://github.com/VocaDB/vocadb/issues/1022))
* Translations are now managed through Crowdin, accessible at [VocaDB Crowdin Project](https://crowdin.com/project/vocadb)

**Fixes:**
* YouTube channel handle URLs are now recognized by the web link matcher ([#1314](https://github.com/VocaDB/vocadb/issues/1314))
* Fixed the "Find Originals" button ([#1369](https://github.com/VocaDB/vocadb/issues/1369))
* Fixed the issue where the overview page of song lists was not linked correctly when exiting a song list ([#1374](https://github.com/VocaDB/vocadb/issues/1374))
* The header on the "Entry Reports" page is now clickable again ([#1376](https://github.com/VocaDB/vocadb/issues/1376))
* Artist information is now displayed correctly ([#1379](https://github.com/VocaDB/vocadb/issues/1379))
* The publish date field now shows the correct text ([#1380](https://github.com/VocaDB/vocadb/issues/1380))
