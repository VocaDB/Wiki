Translations/romanizations of titles and lyrics using [machine translation](https://en.wikipedia.org/wiki/Machine_translation) tools are not allowed. This includes using traditional machine translation services tools such as [Google Translate](https://en.wikipedia.org/wiki/Google_Translate) or [DeepL Translator](https://en.wikipedia.org/wiki/DeepL_Translator), or using [large language models](https://en.wikipedia.org/wiki/Large_language_model).

Official translations are accepted in any case.

It is OK to leave the fields empty for others with experience to fill it for you.
