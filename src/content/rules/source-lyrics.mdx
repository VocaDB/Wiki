Always leave a source for the lyrics; sources should provide a link to an originating source and / or mention the name of the translator. If transcribed, translated, or transliterated by yourself, you should source your own name and provide a relevant link (such as your VocaDB user profile).

If the lyric source is missing, locate it before modifying the lyrics. See [a list of websites where to look for lyrics](/docs/useful-related-informational-sites#song-lyrics). If the source can't be found, create an entry report for the missing source and use a credit like "Unknown (fixed by user2)".
